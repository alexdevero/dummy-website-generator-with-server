# Dummy website generator with Node.js server

Dummy website generator with simple Node.js server

## Installation

```sh
yarn
```

## Run website generator

```sh
yarn start
```

## Start server

```sh
yarn server
```

## Clear generated output

```sh
yarn clean
```
